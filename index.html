About this site

This site is based on the work of Piet Mondrian, a modern artist from the early 1900s responsible for works such as "Composition in Red, Blue, and Yellow, 1930." (pictured on left). This project attempts to reimagine Mondrian's work from a computational perspective.

Utilizing the Wordnik and Flickr APIs Each artwork is generated from a seed word and an associated image. Upon receiving the information, a small color palette is extracted from the image using a slightly modified version of Color Thief. It also utilizes the Google Fonts API to make the text you are reading right now look prettier.

Once the color palette is generated, the site uses Paper.js to randomly generate and color the rectangles. Visit the project's GitHub repository for more information, updates, etc.

Instructions:

  1. Click "Generate" or reload the page to generate art
  2. Click "Details" to show the image and word used for the art
  3. Repeat until bored
